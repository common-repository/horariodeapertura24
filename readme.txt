== Description ==

= ¡ Horarios de apertura en su página web ! =

Contiene más de 1 millón de horarios de apertura de negocios, autoridades públicas, médicos y más.
La instalación es muy fácil. Sólo tiene que ajustar el plugin como lo desee para su página web.

El visitante puede buscar por “qué” y “dónde” y obtiene los resultados directamente en el plugin. Todos los colores del plugin (colores de fondo y texto) pueden ser ajustados libremente. Además, el “qué” y el “dónde” opcionalmente pueden estar predeterminados, por ejemplo “correos”, “Madrid”.

Así nunca más se encontrará ante puertas cerradas 

= Fuente de datos / Participar =
Los datos han sido colectados por 50.000 usuarios del directorio horariodeapertura24 y son actualizados y ampliados continuamente.
¡Naturalmente Usted mismo puede añadir horarios de apertura o editar entradas existentes!

= Comunicación externa =
El widget comunica con nuestro servidor horariodeapertura24.es. En concreto, envía los parámetros de búsqueda a nuestro servidor y obtiene como respuesta los resultados de la búsqueda.

== Installation ==
1. Instalar y activar el plugin
2. Adaptar el diseño del plugin a su página (Ajustes)
2. Añadir el plugin bajo Apariencia->Widgets a la área para widgets.
